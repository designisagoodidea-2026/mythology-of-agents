# The Mythology of Agents

A scroll-driven visual essay on borrowing humanity's oldest framework — mythology — to reason about AI agents: who exists (the chain of agency), what powers they hold (schools of magic), and how an organization decides which powers to grant (the Agency Map).

**Live:** https://designisagoodidea-2026.github.io/mythology-of-agents/

Single self-contained HTML file. Original ink-plate illustrations rendered as inline SVG. No build step, no dependencies beyond Google Fonts.
